Features

  • Speedup your training by auto tuning ML systems confugration

    • Determinie the right set of knob values that achieve optimal completion time without system expertise

  • Hands-free hyperparameter optimization

    • Choose the ideal set of hyperparameters to achive higher model accuracy

  • Easy to use with different ML Systems

    • Provide a simple UI to submit the jobs and monitor the tuning porgress
